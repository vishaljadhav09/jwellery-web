# Mahalaxmi Jwellers 💍

A full-featured e-commerce website for **Mahalaxmi Jwellers**, a silver jewellery shop & manufacturer based in Satara, Maharashtra, India.

## 🌟 Features

- 🛍️ **Product Catalog** – Browse 8+ silver jewellery categories with filters and search
- 🛒 **Shopping Cart** – Persistent cart with Zustand + localStorage
- 💳 **Checkout** – Full checkout flow with Razorpay payment integration
- 👨‍💼 **Admin Dashboard** – Product & order management, analytics (password: `admin123`)
- 🌐 **Bilingual** – Complete English & Marathi (मराठी) support
- ✨ **Animations** – Framer Motion: parallax hero, mandala rotations, staggered reveals
- 📱 **Responsive** – Mobile-first design

## 🎨 Design

Traditional Indian festive aesthetic — deep maroon, gold accents, rangoli patterns, Devanagari typography.

## 🛠️ Tech Stack

- **React 18** + Vite
- **Tailwind CSS** (custom festive theme)
- **Framer Motion** (animations)
- **Zustand** (cart & admin state)
- **react-i18next** (English/Marathi)
- **react-hot-toast** (notifications)
- **Netlify** (free hosting)

## 🚀 Run Locally

```bash
npm install
npm run dev
```
